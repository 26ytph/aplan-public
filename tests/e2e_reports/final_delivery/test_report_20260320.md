## 🧪 測試案例 1：語意標籤與情境聯動檢索 (Semantic UX Integration)

### 測試步驟
1. 導覽至首頁。
2. 等待系統自動獲取 Geolocation 裝置定位與 Open-Meteo 當下氣候。
3. 在動態的情境標籤列表中，手動點擊 **「🏛️ 歷史古蹟」**。
4. 點擊 **「產生我的時光機導覽」**。

### 預期結果
1. 前端應顯示載入中動畫「AI 時光機計算中...」。
2. 後台 Gemini LLM 應正確接收 `intent` (歷史古蹟)，加上目前天氣 (如涼爽/多雲) 與地點。
3. ChromaDB 應從 471 筆 POIs 中檢索出最具歷史意義且地理位置適當的景點。
4. UI 應成功渲染出 1 段總結文案與 3 筆精美的推薦卡片。

### 實際執行結果 ✅ (PASS)
**AI 總結文案生成成功**：
> "今天天氣涼爽舒適，非常適合走訪臺北的歷史古蹟！我為您規劃了一場穿越時空的知性之旅，從日治時期的銀行建築到醫學人文殿堂，讓您在漫步中細細品味臺北深厚的歷史底蘊。"

**推薦清單命中率精準**：
*   **Top 1**: `國立臺灣博物館_古生物館` (AI 評語：這座由勸業銀行改建的古蹟超壯觀！涼爽的天氣最適合在挑高大廳欣賞恐龍化石...)
*   **Top 2**: `國立臺灣大學博物館群_醫學人文博物館` (AI 評語：這棟百年紅磚建築充滿故事...)
*   **Top 3**: `台北當代藝術館` (AI 評語：這座古蹟建築本身就是藝術品...)
